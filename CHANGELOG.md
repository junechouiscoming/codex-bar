# Changelog

## 1.2 - 2026-06-29

### English

- Added reset credit expiry details in a hover table next to the remaining reset count.
- Added token usage notifications after Codex responses.
- Added quota reset reminder notifications for the 5-hour and weekly quota windows.
- Added right-click menu toggles for token and quota notifications.
- Added a generated app icon to the packaged app bundle.

### 中文

- 在剩余重置次数旁新增悬停表格，展示每个重置额度的标题和过期时间。
- 新增 Codex 回答完成后的 token 用量通知。
- 新增 5 小时额度和周额度的重置提醒通知。
- 新增右键菜单中的 token 通知和额度通知开关。
- 为打包后的 app bundle 新增生成式应用图标。

## 1.1 - 2026-06-18

### English

- Added the remaining quota reset credit count below the weekly limit reset time.
- Replaced the monthly token heatmap with a compact daily bar chart.
- Increased panel height and spacing for a calmer layout.
- Refined the liquid-glass panel background and transparency.
- Improved footer alignment for the manual refresh button and last-updated text.

### 中文

- 在周限额重置时间下方新增剩余可用重置次数。
- 将当月 token 热力图改为每日 token 迷你柱状图。
- 增加面板高度和内部间距，让布局更舒展。
- 调整 liquid glass 面板背景和透明度。
- 优化右下角手动刷新按钮和更新时间的对齐。
